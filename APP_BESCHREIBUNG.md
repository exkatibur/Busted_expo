# BUSTED! - Where friends become traitors

## Was ist das?

BUSTED! ist DAS Party-Spiel für dich und deine Freunde. Ihr sitzt zusammen rum und müsst euch gegenseitig "verraten" - also abstimmen, wer von euch am ehesten bestimmte Sachen macht oder gemacht hat. Am Ende wird enthüllt, wen alle gewählt haben. Awkward? Definitiv. Lustig? 100%!

---

## Features im Detail

### 1. Account & Login

- **Anonymer Start**: Kein nerviges Registrieren mit E-Mail. Einfach App öffnen und loslegen.
- **Username wählen**: Beim ersten Start gibst du dir einen coolen Namen (z.B. "ChaosMaster" oder "SarahXD")
- **Optional: Account upgraden**: Wenn du willst, kannst du später eine E-Mail verknüpfen, um deinen Account auf anderen Geräten zu behalten
- **Ausloggen**: Jederzeit möglich über den Home-Screen

### 2. Raum erstellen (Host)

- **Raum-Code generieren**: Du bekommst automatisch einen einzigartigen 6-stelligen Code (Buchstaben und Zahlen, z.B. "AB4K7M")
- **Code teilen**: Der Code kann mit einem Tap kopiert werden, um ihn easy an Freunde zu schicken
- **Du bist der Host**: Als Ersteller hast du die Kontrolle über das Spiel
- **Host-Badge**: Im Spiel sehen alle, dass du der Host bist (kleines Symbol neben deinem Namen)

### 3. Raum beitreten

- **Code eingeben**: Einfach den 6-stelligen Code eintippen, den dir dein Freund geschickt hat
- **Automatisch Großbuchstaben**: Egal ob du groß oder klein schreibst, der Code wird erkannt
- **Fehler-Feedback**: Wenn der Code falsch ist oder der Raum nicht existiert, bekommst du eine Meldung

### 4. Lobby (Warteraum)

- **Raum-Code groß anzeigen**: Der Code wird prominent angezeigt, damit Nachzügler ihn sehen können
- **Kopier-Button**: Ein Button neben dem Code zum schnellen Kopieren
- **Teilnehmer-Liste**: Zeigt alle, die schon im Raum sind
  - Jeder Spieler wird mit Username angezeigt
  - Der Host hat ein spezielles Symbol
  - Neue Spieler erscheinen sofort in Echtzeit (ohne Refresh)
- **Spieler-Zähler**: Zeigt "X / 30 Spieler" an
- **Start-Button**: Nur der Host sieht den "SPIEL STARTEN" Button
- **Mindestens 2 Spieler**: Das Spiel kann erst starten, wenn mindestens 2 Leute da sind
- **Verlassen**: Jeder kann den Raum verlassen (mit Bestätigungs-Dialog)
- **Host verlässt = neuer Host**: Ein beliebiger anderer Spieler wird Host

### 5. Vibe auswählen (Spielmodus)

Vor dem Spielstart wählt der Host einen "Vibe" - das bestimmt, welche Art von Fragen kommen:

- **Party**: Klassische Party-Fragen für jeden Anlass
- **Date Night**: Romantischere Fragen für Pärchen oder flirty Abende
- **Family**: Familienfreundliche Fragen (keine peinlichen Themen)
- **Spicy**: Freche, pikante Fragen für Mutige

### 6. Das Spiel - Fragen & Abstimmung

#### Frage anzeigen
- **Frage-Karte**: Die Frage wird groß und zentral angezeigt
- **Frage-Typen**:
  - "Wer würde am ehesten..." (Most likely to)
  - "Wer hat schon mal..." (Never have I ever - aber umgedreht)
- **Runden-Anzeige**: Oben steht, welche Runde gerade läuft (z.B. "Runde 3")

#### Abstimmen
- **Spieler-Liste**: Unter der Frage siehst du alle Mitspieler
- **Einen auswählen**: Du tippst auf die Person, die deiner Meinung nach am besten zur Frage passt
- **Du kannst dich NICHT selbst wählen**: Dein eigener Name ist nicht auswählbar
- **Einmal gewählt = fix**: Du kannst deine Wahl nicht mehr ändern
- **Bestätigung**: Nach dem Tap kommt kurz "Abgestimmt!" als Feedback

#### Warte-Phase
- **Status-Anzeige**: "Warte auf andere Spieler..."
- **Vote-Counter**: Zeigt "X von Y haben abgestimmt" (z.B. "5 von 8 haben abgestimmt")
- **Echtzeit-Update**: Der Counter aktualisiert sich live, wenn jemand abstimmt

#### Reveal erzwingen (nur Host)
- **Reveal-Button**: Der Host sieht einen Button "ERGEBNISSE ZEIGEN"
- **Ab 50% aktiv**: Der Button wird erst klickbar, wenn mindestens die Hälfte abgestimmt hat
- **Bestätigungs-Dialog**: "Nur X von Y haben abgestimmt. Trotzdem aufdecken?"
- **Countdown**: Wenn alle abgestimmt haben ODER der Host auf Reveal drückt, startet ein 5-Sekunden-Countdown

### 7. Ergebnis-Screen (BUSTED!)

#### Enthüllung
- **"BUSTED!" Überschrift**: Groß und dramatisch
- **Konfetti-Explosion**: Buntes Konfetti regnet von oben
- **Podium-Anzeige**: Die Top 3 werden wie bei einer Siegerehrung gezeigt
  - Platz 1 (Mitte, am höchsten): Die Person mit den meisten Stimmen
  - Platz 2 (links): Zweite meiste Stimmen
  - Platz 3 (rechts): Dritte meiste Stimmen
- **Stimmen-Anzahl**: Bei jedem Namen steht, wie viele Stimmen sie bekommen haben

#### Vollständige Ergebnis-Liste
- **Alle Spieler**: Unter dem Podium werden ALLE Spieler aufgelistet
- **Sortiert nach Stimmen**: Von den meisten zu den wenigsten
- **Wer hat wen gewählt**: Man kann sehen, wer für wen gestimmt hat (Transparenz!)

#### Host-Kontrollen
- **"NÄCHSTE RUNDE" Button**: Startet die nächste Frage
- **"SPIEL BEENDEN" Button**: Beendet das Spiel und schickt alle zurück zur Lobby

### 8. Session-Recovery (Wiedereinstieg)

- **App geschlossen?**: Wenn du die App schließt und wieder öffnest, landest du automatisch wieder im Spiel
- **Internet weg?**: Bei kurzen Verbindungsproblemen bleibst du im Spiel
- **Zurück im richtigen Screen**: Du landest genau da, wo das Spiel gerade ist (Lobby, Frage oder Ergebnis)

### 9. Spieler verlässt während des Spiels

- **Bestätigungs-Dialog**: "Willst du wirklich gehen?"
- **Spiel geht weiter**: Die anderen können weiterspielen
- **Stimme zählt nicht**: Wenn jemand geht, bevor er abgestimmt hat, wird er nicht gezählt
- **Host geht = neuer Host**: Wenn der Host während des Spiels geht, wird ein beliebiger anderer Spieler Host

## Spieler-Limits

- **Minimum**: 2 Spieler
- **Maximum**: 30 Spieler pro Raum
- **Raum voll**: Wenn 30 Leute drin sind, können keine weiteren beitreten (Fehlermeldung)
